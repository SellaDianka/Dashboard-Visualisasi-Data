# Dashboard Indikator Makro Sosial Ekonomi Provinsi Lampung

## Deskripsi
Dashboard ini merupakan visualisasi data triwulanan indikator makro sosial ekonomi Provinsi Lampung yang dikembangkan menggunakan Power BI. Dashboard dirancang untuk membantu proses pengelolaan, analisis, dan penyajian data secara interaktif sehingga informasi dapat dipahami dengan cepat dan efektif.

## Cakupan Indikator
Dashboard menampilkan indikator utama:
- Produk Domestik Regional Bruto (PDRB)
- Inflasi
- Neraca Perdagangan Luar Negeri
- Ekspor
- Impor
- Nilai Tukar Petani (NTP)

## Tujuan
- Memudahkan pemantauan perkembangan indikator ekonomi daerah
- Menyediakan visualisasi data yang interaktif dan terstruktur
- Mengidentifikasi tren dan pola indikator dalam beberapa tahun terakhir
- Mendukung analisis data dan pengambilan keputusan

## Manfaat
Penggunaan Power BI memungkinkan pengguna:
- Mengeksplorasi data secara interaktif
- Memperoleh insight dengan cepat
- Membandingkan indikator antar periode
- Mendukung perencanaan dan evaluasi kebijakan

## Kesimpulan
Dashboard visualisasi indikator makro sosial ekonomi Provinsi Lampung berbasis Power BI terbukti efektif sebagai alat bantu dalam menampilkan, memantau, dan menganalisis data ekonomi daerah secara interaktif untuk mendukung pengambilan keputusan.
